## Getting started

<Steps>
  <Step title="Get a Gemini API key">
    Sign up at [Google AI Studio](https://aistudio.google.com) and create a free API key.
  </Step>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/pwobus/Mandarin-Talk.git
    cd Mandarin-Talk
    npm install
    ```
  </Step>
  <Step title="Add your API key">
    Create `.env.local` in the project root and add:
    ```bash .env.local
    GEMINI_API_KEY=your_api_key_here
    ```
  </Step>
  <Step title="Run the app">
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser, allow microphone access, and press **Start Conversation** to begin your first conversation.
  </Step>
</Steps>

<Note>
  Mandarin Talk requires microphone access and a modern browser with Web Audio API support (Chrome or Edge recommended).
</Note>
