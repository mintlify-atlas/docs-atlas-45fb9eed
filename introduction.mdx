Mandarin Talk gives Mandarin learners an always-available, low-pressure environment to build fluency. Speak naturally into your microphone, and an AI partner responds in real time — coaching your pronunciation, providing layered subtitles, and keeping the conversation going at your pace.

## How it works

Mandarin Talk runs entirely in the browser. When you press **Start**, it opens a live audio session with [Google Gemini](https://aistudio.google.com) using the Gemini Live API (`gemini-2.5-flash-native-audio-preview`). Your microphone audio is streamed continuously to the model, which responds with spoken Mandarin and simultaneously calls structured tools to produce subtitles and pronunciation scores — all with minimal latency.

No server of your own is needed. The app reads your `GEMINI_API_KEY` from a local `.env.local` file and connects directly to the Gemini API from the browser.

<Note>
  Mandarin Talk uses the Web Audio API for microphone capture and real-time audio playback. Chrome and Edge provide the most consistent experience.
</Note>

## Prerequisites

Before you run Mandarin Talk, make sure you have the following:

- **Node.js** (v18 or later) — required to install dependencies and run the Vite dev server
- **A modern browser** — Chrome or Edge recommended; both support the Web Audio API and WebSocket connections required by the Gemini Live API
- **A microphone** — built-in or external; the app requests microphone access when you press Start
- **A Gemini API key** — free to create at [Google AI Studio](https://aistudio.google.com)
