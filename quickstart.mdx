<Steps>
  <Step title="Get a Gemini API key">
    Mandarin Talk connects directly to the Gemini Live API from your browser. You need a free API key to authenticate.

    1. Go to [Google AI Studio](https://aistudio.google.com).
    2. Sign in with a Google account.
    3. Click **Get API key** and create a new key.
    4. Copy the key ÔÇö you'll add it to your environment in a later step.

    <Note>
      The free tier of the Gemini API is sufficient for personal practice sessions. No billing setup is required to get started.
    </Note>
  </Step>

  <Step title="Clone the repository">
    Clone the Mandarin Talk source code to your local machine:

    ```bash
    git clone https://github.com/pwobus/Mandarin-Talk.git
    cd Mandarin-Talk
    ```
  </Step>

  <Step title="Install dependencies">
    Install the required Node.js packages:

    ```bash
    npm install
    ```

    This installs React 19, Vite, the `@google/genai` SDK, Three.js for the 3D avatar, and all other dependencies listed in `package.json`.
  </Step>

  <Step title="Add your API key">
    Create a `.env.local` file in the project root and add your Gemini API key:

    <CodeGroup>

    ```bash .env.local
    GEMINI_API_KEY=your_key_here
    ```

    </CodeGroup>

    The app reads this value as `process.env.API_KEY` at runtime. Vite exposes environment variables prefixed with `VITE_` by default, but this project uses a custom Vite configuration to expose `API_KEY` from `GEMINI_API_KEY`. The `LiveApiService` picks it up here:

    ```typescript services/liveApiService.ts
    this.ai = new GoogleGenAI({ apiKey: process.env.API_KEY });
    ```

    <Note>
      Never commit `.env.local` to version control. The file is already listed in `.gitignore`.
    </Note>
  </Step>

  <Step title="Start the dev server">
    ```bash
    npm run dev
    ```

    Vite will print a local URL ÔÇö typically `http://localhost:5173`. Open it in Chrome or Edge.
  </Step>

  <Step title="Open the app, allow microphone access, and press Start">
    1. Open the URL printed by Vite in your browser.
    2. When prompted, click **Allow** to grant microphone access. The app cannot connect to Gemini Live without it.
    3. Press the **Start** button in the center of the screen.

    The status indicator in the top-right corner will cycle from `DISCONNECTED` Ôćĺ `CONNECTING` Ôćĺ `CONNECTED`.
  </Step>
</Steps>

## Your first conversation

Once the status shows `CONNECTED`, Mandarin Talk is live:

- **The AI starts listening immediately.** Speak into your microphone ÔÇö you don't need to press any button to talk.
- **Try saying anything.** You can start in English and the AI will gently steer the conversation toward Mandarin. Or jump straight in: ńŻáňąŻ´╝ü(*NăÉ hăÄo!*)
- **Subtitles appear in real time.** Every sentence the AI speaks is shown in three layers ÔÇö Hanzi, Pinyin, and English ÔÇö in the center panel.
- **After you speak Mandarin, a pronunciation score card appears** on the right side of the screen. It shows the text you said, its Pinyin, a 0ÔÇô100 score, and specific feedback on your tones or articulation.
- **Your conversation is logged** in the history panel on the left (desktop) so you can review every turn after the session.

To end the session, press **Stop**. To start a new conversation, press **Start** again ÔÇö your history is cleared on each new connection.

<Tip>
  Try selecting a lesson from the **Library** (top-right) before pressing Start. Lessons like "The Coffee Shop" or "Market Bargaining" give the AI a structured roleplay scenario, which makes it easier to practice specific vocabulary and stay on topic.
</Tip>
